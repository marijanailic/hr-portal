### Setup Instructions

1. Clone repo

    git clone https://github.com/YOUR_USERNAME/hr-portal.git

    cd hr-portal

2. Client setup http://localhost:3000

    cd client

    npm install

    npm start

3. Server setup http://localhost:5000
    cd server

    npm install

    npm start
